Flexbox

Permite posicionamentos elementos dentro das caixas. Porém para que um elemento desfrute dos atributos do flexbox, o elemento pai deve receber a seguinte configuração:

             
                 `````css
                            div.parent{
                                display: flex;
                            }
                 `````

Flex-direction:

  • row
  • column

Alinhamento:

  • justify-content
  • align-items
