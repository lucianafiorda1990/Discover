Button

  • Representa um botão
  • Usado para enviar formulários
  • É estilizado pelo user agent

Atributos comuns:

  • type
    • submit - envia (submete) os dados preenchidos no formulário
    • reset - limpa os dados preenchidos no formulário
    • button - alguns navegadores também entendem que é para submeter o form com este tipo de botão
  • autofocus - é como se já estivesse selecionado o botão
  • disabled - desabilita o botão
  • name - dá um nome ao botão
  • value - seta um valor para o botão e esse valor vai para outro elemento com mesmo nome
  • form - linka uma botão a um formulário através do ID do formulário
