Datalist

  • Lista de valores como sugestão a uma tag <input>
  • Valores sugestivos e não obrigatórios
  • Usuário poderá selecionar um dos valores, ou colocar um valor diferente da seleção
  • O link é feito através do atributo "link"

OBS: se caso o campo for datalist porém não receber um dado no formato correto, ele irá ignorar a lista. Por exemplo: um input do tipo "color" precisa dos itens da lista no formato código rgb.


Tipos de input que suportam datalist e list:

  • text, search, url, tel, email, date, month, week, time, datetime-local, number, range e color.

    • Tipos de input que não suportam datalist e list:

  • hidden, password, checkbox, radio file ou qualquer tipo de button.
