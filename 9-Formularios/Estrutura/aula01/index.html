Aula 01

Formulários

Servem para capturar dados do usuário. São feitos através da tag <form>.

Seus atributos básicos são:

  • action: informa para onde os dados devem ser enviados. Se caso não tiver este atributo ou ele ficar em branco, irá enviar para a mesma página de origem.
  • method: maneira com que os dados serão enviados. Temos a opção de enviar via método GET ou POST.

Exemplo de um método GET:

        
            http://minhapagina.com/?name=Luciana&email=luzinha_fiorda@hotmail.com

OBS: Se não informarmos o método, irá ser aplicado o método padrão GET.

OBS: O método POST não deixa os dados expostos como o método GET, por isso é bem mais usado principamente na criação da cadastros.

OBS: Nunca fazer um form dentro de outro form.
