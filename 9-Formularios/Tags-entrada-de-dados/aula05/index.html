Aula 05

URL

Espera que o usuário digite uma URL e vai validar se está no formato correto.

Atributos

  • placeholder
  • readonly
  • disabled
  • value
  • required
  • min/maxlength
  • pattern
  • size
  • list: para linkar com a <datalist>
  • title
  • spellcheck: habilita a verificação ortográfica do campo. Recebe valores como true/ false.
