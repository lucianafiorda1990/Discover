Aula 11

Textarea

Usado para textos grandes.

Atributos

  • id: para linkar com o label
  • name: para quando for enviar os dados
  • rows e cols: quantidade de linhas e colunas que vão definir o tamanho da box de texto
  • maxlength e minlength: máximo e mínimo de caracteres
  • wrap: é referente a quebra de linha. Recebe os valores hardsoft/on e off. Por padrão é on, e faz a quebra automática de linhas. Se colocarmos off, fica tudo numa linha só e com barra de rolagem horizontal.
  • Outros atributos comuns de input: autocomplete, autofocus, disabled, placeholder, readonly, form, required...
