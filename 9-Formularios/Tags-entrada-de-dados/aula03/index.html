Password

Campo destinado a colocação de senhas. É seguro, pois esconde o que foi digitado. O estilo padrão do campo depende do user agent (navegador).

Atributos:

  • minlenght: determina uma quantidade mínima de caracteres no campo
  • maxlenght: determina uma quantidade máxima de caracteres no campo
  • size: determina o tamanho do campo (caixa branca de fundo)
  • pattern: expressão regular que para validar o que está sendo digitado no campo. Exemplo: se quisermos uma senha hexadecimal entre 4 e 8 dígitos: pattern="[0-9a-fA-F]{4,8}"
  • title: coloca um título no campo que irá aparecer em tela
  • placeholder: texto de exemplo
  • readonly: nao permite alterações no campo
  • required: torna o campo de preenchimento obrigatório
  • inputmode: configura como irá abrir o teclado no celular para inserir a senha. Exemplo: inputmode="numeric" (só irá abrir o tecldo numérico do celular)
  • autocomplete: permite o autocomplete/ sugestão de senhas
    • on: permite a sugestão de new-password ou current-password
    • off: desabilita a opção de autocompletar
    • new-password: o navegador poderá sugerir uma nova senha
