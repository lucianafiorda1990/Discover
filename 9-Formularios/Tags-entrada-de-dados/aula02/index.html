Input - atributos

  • autocomplete: auto completa com sugestões, algo ja muito digitado no navegador
  • autofocus: deixa já selecionado o campo para ser digitado - só pode ter um autofocus na página
  • disabled: para desabilitar o campo
  • readonly: permite apenas visualização e não alteração
  • value: já deixa um valor pré definido já para o campo
  • form: serve para linkar um input a um formulário
  • name: serve para nomear o input e será usado para coletar os dados do campo
  • required: torna o campo de preenchimento obrigatório
  • placeholder: deixa algo já digiado dentro do campo, para ajudar o usuário. Pode substituir a tag <label>, porém reduz a acessibilidade.
