File

Permite ao usuário escolher um ou mais arquivos para enviar no formulário.

Atributos

  • value: contém o arquivo a ser enviado
  • accept: descreve os tipos de arquivos que serão aceitos. Exemplo: accept=".doc, image/png, audio/*" (o * representa todos os tipos de audio no exemplo antetior.)
  • multiple: permite o envio de vários arquivos
  • files: lista de arquivo(s)
Quando fazemos um input do tipo file, o form precisa ter o "post" como "method" e o atributo enctype="multipart/form-data".
