Box Model

Box Model é fundamental para fazer layouts para a web e facilita a aplicação do CSS.

O que é? uma caixa retangular com propriedades.

Propriedades:
Propriedade Property
Tamanho (largura x altura) Width x Height
Conteúdo Content
Bordas Border
Preenchimento interno Padding
Espaços fora da caixa Margin

Caa elemento na página é uma caixa!
