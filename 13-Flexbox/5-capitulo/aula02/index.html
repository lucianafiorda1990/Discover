Flex-basis

Se estivermos com o flex-basis: auto, irá considerar o width/ height passado no item. Se ele estiver com um valor, será considerado este valor. Mesmo que depois seja passado um valor específico para cada item filho.

Quando estamos com o flex-direction: row o flex-basis muda o width. Porém se estivermos com o flex-direction: column irá mudar o height.
