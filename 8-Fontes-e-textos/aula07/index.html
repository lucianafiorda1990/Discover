Web-fonts

Podemos usar fontes do sistema (local) ou da web

Como usar fontes da web?

  • @font-face
  • @import
  • link

Exemplo de @font-face:

        
            /* vietnamese */
            @font-face {
            font-family: 'Licorice';
            font-style: normal;
            font-weight: 400;
            font-display: swap;
            src: url(https://fonts.gstatic.com/s/licorice/v1/t5tjIR8TMomTCAyjNn2_jKPQzA.woff2) format('woff2');
            unicode-range: U+0102-0103, U+0110-0111, U+0128-0129, U+0168-0169, U+01A0-01A1, U+01AF-01B0, U+1EA0-1EF9, U+20AB;
            }

Exemplo de @import (deve ser colado no head dentro da tag <style>):

        
            style
                @import url('https://fonts.googleapis.com/css2?family=Licorice&family=Twinkle+Star&display=swap');
            /style

Exemplo de link (deve ser colado no head):

        
            link rel="preconnect" href="https://fonts.googleapis.com"
            link rel="preconnect" href="https://fonts.gstatic.com" crossorigin
            link href="https://fonts.googleapis.com/css2?family=Licorice&family=Twinkle+Star&display=swap" rel="stylesheet"
