SVG

<svg> é uma tag usada na criação de imagens.

Imagem Rasterizada x Vetorizada

  • Rasterizada: é a imagem já pronta formada por milhões de pixels.
  • Vetorizada: é a imagem formada através de códigos (SVG)

Benefícios da imagem SVG

  • Mais leve e detalhada
  • Maior acessibilidade e SEO (mecanismos de busca)
  • Pode ser editada via CSS ou atributos

Desvantagens da imagem SVG

  • Pode ser mais complicado de trabalhar
  • quanto mais complexa a imagem, mais trabalho para o navegador
  • Navegadores mais antigos não suportam a tag <svg>

Para fotografias, prefira usar imagens rasterizadas.

Exemplos de imagens rasterizadas: .png / .jpg / .jpeg
