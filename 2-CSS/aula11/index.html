Shorthand

É um conceito de junção de propriedades no CSS com intuito de ser mais resumido e legível.

Exemplo:

Modo normal:

            
                background-color: #000;
                background-image: url(images/bg.gif);
                background-repeat: no-repeat;
                background-position: left top;

Modo shorthand:

            
                background: #000; url(images/bg.gif) no-repeat left top;

OBS: valores não especificados, irão assumir valor padrão

OBS 2: geralmente a ordem descrita não importa, porém podemos ter problemas
