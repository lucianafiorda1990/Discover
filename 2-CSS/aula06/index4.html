Origem do CSS

Importando um arquivo da Internet

Este é um exemplo de um parágrafo com estilização importando o arquivo do Google Fonts.

O arquivo pode ser importado diretamente na tag <style> na <head> ou no arquivo style.css separado do HTML.

Podemos importar diretamente pela tag <link> no <head> também. É mais indicado fazer assim inclusive pois é mais rápido o carregamento.

Exemplo:

        
            link rel="preconnect" href="https://fonts.googleapis.com"
            link rel="preconnect" href="https://fonts.gstatic.com" crossorigin
            link href="https://fonts.googleapis.com/css2?family=Twinkle+Star&display=swap" rel="stylesheet"
