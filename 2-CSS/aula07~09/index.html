Cascata

O estilo é lido de cima para baixo, levando 3 fatores em consideração:

OBS: se caso tiver dois links style.css, irá considerar o último importado.

  1. Origem do estilo: onde está sendo atribuido o estilo

    2. inline > tag style > tag link

  2. Especificidade: é feito um cálculo matemático, onde é considerado o tipo de seletor e a origem do estilo.
    • 0. Universal selector, combinators e negation pseudo-class (:not())
    • 1. Element type selector e pseudo-elements (::before, ::after)
    • 10. Class e attribute selectors ([type="radio"])
    • 100. ID selector
    • 1000. Inline
  3. !important: se colocar essa expressão na estilização, ela irá sobrescrever qualquer outra já usada, mesmo a mais forte
