Aula 12

Funções

Tarefas que vão executar certas funcionalidades, recebem parâmetros.
Exemplos:

        
            @import url("link");

            /* OU */

            {
                color: rgb(255,0,100);
                width: calc(100% - 10px);
            }
  • URL: retorna um link
  • RGB: retorna uma cor
  • CALC: retorna um calculo matemático
