Background-image



Se a imagem escolhida para o fundo for menor que o espaço disponível, ela automaticamente irá se repetir para preencher todo o espaço.



Background-repeat

  • no-repeat: faz a imagem não se repetir nenhuma vez
  • repeat-y: faz a imagem se repetir no eixo Y (verticalmente)
  • repeat-x: faz a imagem se repetir no eixo x (horizontalmente)
