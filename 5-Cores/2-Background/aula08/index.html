Aula 08

Background shorthands

Podemos colocar todas as configurações do backgroud em apenas uma linha no CSS.

        
            backgroud: rgb(50,30,20)    url(link)    no-repeat      right top      /        50px       border-box       content-box       fixed;
                           color          image        repeat       position                size        origin             clip         attachment

OBS: É obrigatório ter a barra entre o position e o size.

OBS 2: se colocar apenas umas config do tipo origin / clip irá se aplicar aos dois a mesma configuração. Porém se forem informados dois valores como no exemplo acima, o primeiro será para o origin e o segundo para o clip.
