Strings

Temos 3 maneiras de escrever strings:

  • Aspas simples Ex: 'Luciana'
  • Aspas duplas Ex: "Luciana"
  • Crase Ex: `Luciana`

O método da crase também é conhecido como Template Literals ou Template String.

Ele também permite o texto multilinhas e o uso de aspas (simples e duplas) tranquilamente no meio do texto sem gerar erros.

Também podemos usar expressões e variáveis dentro do console.log com o uso de ${expressao}. Isso é conhecido como interpolação.

Exemplo

        
            console.log(`1 + 1 = ${1 + 1}`)
            console.log(`1 + 1 = ${name}`)
