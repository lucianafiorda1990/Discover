Aula 01

Variáveis

Podemos declarar as variáveis de 3 maneiras:

  • var
  • let
  • const

Sendo que const é relativo a constante, então uma vez atribuido um valor a variável, ele não pode ser alterado. 

        
            let name = 'Luciana'
            var age = 31
            const eyes = 'Brown'
