Aula 04

Javascript - primeiros passos

Podemos linkar o Javascrip com o HTML de 2 maneiras:

  • Pela tag <script> no <body>
  • Ou fazer um arquivo separado no formato .js e linkando ele através da tag <script src="localdoarquivo.js">

Comentários

Temos dois tipos de comentário:

  • De linha através do símbolo //comentário
  • Ou em bloco através do símbolo /* comentário */

Exemplo

        
            
                // comentário de uma linha

                /*

                    comentário 
                    de bloco

                */
